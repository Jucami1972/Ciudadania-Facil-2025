# Prompt para Consulta de IA: Mejores Prácticas de Onboarding

## Contexto del Proyecto

Soy desarrollador de una aplicación móvil y web llamada **"Ciudadanía Fácil 2025"**, diseñada para ayudar a personas a prepararse para el examen de ciudadanía estadounidense. La app incluye:

- **Tarjetas de estudio interactivas** (flip cards) con las 128 preguntas oficiales
- **Módulos de práctica** con diferentes modos (por categoría, aleatorio, enfocado en errores)
- **Entrevista simulada con IA** que replica una entrevista real usando el formulario N-400
- **Sistema de progreso y seguimiento** con rachas de estudio y métricas
- **Vocabulario y memoria fotográfica** para aprender conceptos clave

**Tecnología:** React Native con Expo (iOS, Android, Web)
**Audiencia objetivo:** Personas que se preparan para el examen de ciudadanía estadounidense (principalmente hispanohablantes)

---

## Tu Rol y Experticia

Eres un **experto en UX/UI y diseño de experiencias de usuario**, específicamente especializado en:

1. **Diseño de flujos de onboarding** para aplicaciones móviles y web
2. **Psicología del usuario** y principios de engagement
3. **Mejores prácticas de UX** para aplicaciones educativas
4. **Diseño de interfaces** que maximizan la retención de usuarios
5. **Copywriting** para pantallas de onboarding efectivas
6. **Análisis de conversión** y optimización de flujos de usuario

## Tu Tarea

Necesito tu ayuda para diseñar y optimizar un **flujo de onboarding de 5 pantallas** para mi aplicación. Actualmente tengo un componente básico que muestra 5 pasos, pero necesito:

1. **Evaluar el contenido actual** y sugerir mejoras
2. **Proponer la estructura ideal** de las 5 pantallas
3. **Recomendar mejores prácticas** específicas para apps educativas
4. **Sugerir contenido específico** (títulos, descripciones, CTAs) para cada pantalla
5. **Identificar elementos visuales** que deberían incluirse (iconos, ilustraciones, animaciones)
6. **Recomendar estrategias de engagement** para mantener al usuario interesado

---

## Objetivo Principal

Crear un **onboarding efectivo** que:

- ✅ **Eduque** al usuario sobre las funcionalidades principales de la app
- ✅ **Motive** al usuario a comenzar a usar la app inmediatamente
- ✅ **Reduzca la fricción** y el tiempo de aprendizaje
- ✅ **Aumente la retención** de usuarios nuevos
- ✅ **Establezca expectativas claras** sobre qué puede lograr con la app
- ✅ **Sea culturalmente apropiado** para una audiencia hispanohablante que busca ciudadanía estadounidense

---

## Cómo Debes Actuar

1. **Sé específico y accionable**: No solo digas "mejora el contenido", sino proporciona ejemplos concretos de títulos, descripciones y CTAs.

2. **Considera el contexto**: Recuerda que los usuarios están en un proceso emocional importante (prepararse para ciudadanía), así que el tono debe ser:
   - Motivacional pero no exagerado
   - Profesional pero accesible
   - Alentador y empoderador

3. **Piensa en la progresión**: Las 5 pantallas deben tener una narrativa coherente que guíe al usuario desde la introducción hasta la acción.

4. **Sé práctico**: Considera limitaciones técnicas (React Native, diferentes tamaños de pantalla) y sugiere soluciones implementables.

5. **Incluye métricas de éxito**: Si es posible, sugiere cómo medir la efectividad del onboarding.

6. **Proporciona múltiples opciones**: Para cada pantalla, sugiere 2-3 variantes de contenido para que pueda elegir la que mejor se adapte.
